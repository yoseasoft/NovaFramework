## 模块名称：每日特惠商店

### 功能点
1. 商店界面 ：显示可购买的商品列表，包含商品图标、名称、价格、剩余购买次数
2. 每日刷新 ：商店商品每天凌晨自动刷新，重置购买次数
3. 商品购买 ：玩家可以使用游戏货币购买商品，购买后扣除相应货币并减少购买次数
4. 限购机制 ：每种商品有每日购买上限，达到上限后当天无法再次购买
5. 商品分类 ：商店商品按类型分类显示（如资源类、装备类、材料类等）
### 实体定义
1. 商品界面对象 ：根据对应的UI资源创建视图对象
2. 商品信息 ：将商品信息属性封装为组件对象，以管理商品信息数据为主
3. 商品刷新 ：剥离商品刷新业务单独封装为组件对象，以处理商品刷新逻辑为主
4. 商品分类 ：剥离商品分类界面刷新业务单独封装为组件对象，以处理分类界面刷新逻辑为主
### 数据需求
1. 商店界面美术资源 ：商店界面UI，物品图标组件UI
2. 商店配置表 ：
   - 商店货架表 ：商店货架标识，货架序号，货架开启条件
   - 物品表 ：物品ID、名称、图标、类型、物品描述
3. 商店网络协议：
   - 商品列表（下行消息） ：商品ID、商店货架标识、价格、每日限购数量
   - 商品购买（上行消息） ：商品ID、商品数量
### 实现要点
- 每日重置机制需要在客户端实现，确保所有玩家在同一时间刷新
- 购买次数需要持久化存储，确保玩家下线后数据不丢失
- 商店界面需要实时显示剩余购买次数，提供良好的用户体验
