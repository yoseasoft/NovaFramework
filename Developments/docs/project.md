## 项目规则

从 `[智能体目录]/consensus.md` 读取当前状态，根据当前状态执行对应动作：

### 状态：分析阶段

- 参考 `[智能体目录]/docs/development-specs/` 中的开发规范进行提案编写
- 确保提案与 `[智能体目录]/docs/api/` 中的API文档保持一致
- 考虑使用 `[智能体目录]/examples/` 中的示例代码作为参考

### 状态：设计阶段

- 基于 `[智能体目录]/docs/development-specs/` 中的开发规范进行设计
- 参考 `[智能体目录]/docs/api/` 中的接口定义
- 利用 `[智能体目录]/examples/` 中的示例代码作为实现参考

### 状态：开发阶段

- 参考 `[智能体目录]/examples/` 中的示例代码进行任务分解
- 与 `[智能体目录]/docs/api/` 中的API文档保持一致
- 确保任务实现符合 `[智能体目录]/docs/development-specs/` 中的开发规范
